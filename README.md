# Flashcard App

AI-powered flashcard generator. Paste notes and generate study flashcards


## Live Demo

[flashcard-app](https://flashcard-app-blond-nine.vercel.app)
## Features

- Generate flashcards from notes using AI (Gemini 2.5 Flash)
- Study mode with Got It / Still Learning tracking
- Decks saved locally in your browser
